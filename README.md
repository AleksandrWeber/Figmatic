## English

### What is Figmatic?
**Figmatic** is an advanced AI-powered **VS Code Extension** that transforms Figma designs into production-ready React code. It acts as a **Senior Frontend Architect**, interpreting design intent and planning modular, testable, and documented component hierarchies.

### Key Features ‚ú®
- **üé® Multi-Style Architecture**: Support for both **Tailwind CSS** and **BEM-styled SCSS**.
- **üî° Interactive Font Intelligence**: Automated commercial font detection with AI-powered Google Font suggestions and manual override dialogue.
- **üìö Storybook Automation**: Instant setup of `.storybook` and automatic `*.stories.tsx` generation for every component.
- **üß™ Vitest Unit Testing**: Automated generation of `*.test.tsx` files and a functional testing environment (Vitest + JSDOM).
- **üèóÔ∏è Professional Bootstrapping**: Generates a complete **Vite + React + TypeScript** project with one click.
- **üöÄ Git Initializer**: Automatic `git init` and professional `.gitignore` setup for every project.
- **‚è±Ô∏è Live Progress & Timer**: Track every AI action in real-time with an active countdown/log.
- **‚ú® AI Post-Generation Loop**: Technical suggestions for performance, accessibility (A11y), and feature improvements.
- **üìÇ "Open Folder" UX**: Direct transition from generation to your new project workspace.

## üöÄ Quick Start

1. Install the extension.
2. Go to **VS Code Settings** and search for `Figmatic`. Enter your `Gemini API Key`.
3. Choose your preferred **Style Framework** (Tailwind or SCSS) in settings.
4. Open the **Figmatic Sidebar** in the Activity Bar.
5. Paste your **Figma URL**.
6. Click **Generate** and watch your architecture take shape!
7. Once finished, click **Open Folder**, run `npm install`, and you're ready to code.

### Technical Setup üõ†Ô∏è

#### 1. Installation
- Clone the repository: `git clone https://github.com/AleksandrWeber/Figmatic.git`
- Install dependencies: `npm install`

#### 2. Configure API Keys
- **Gemini AI**: Open VS Code Settings (`Cmd+,`), search for "Figmatic", and enter your **Gemini API Key**.
- **Figma Token**: Enter it directly in the extension sidebar for the session.

#### 3. Development Mode
- Open the project in VS Code and press **F5**.
- In the new [Host] window, open the **Figmatic** icon in the sidebar.
- Run `npm run watch` (if applicable) and enjoy the AI-driven workflow.
